## Structure

```
thesis/
├── meetings/                      # Weekly supervisor meetings
│   └── YYYY-MM-DD/               # One folder per meeting
│       ├── agenda.md
│       ├── notes.md
│       └── action_items.md
├── literature/                    # Research papers
│   ├── papers/                   # PDFs with naming convention
│   ├── notes/                    # Paper summaries
│   └── PAPER_NAMING.md           # Naming format
├── experiments/                  # Code & analysis
│   ├── 01_dataset/
│   ├── 02_baseline/
│   ├── 03_binary_pso/
│   └── results/
├── data/                         # Datasets
│   ├── raw/
│   └── processed/
├── manuscript/                   # Thesis chapters
└── templates/                    # Templates for meetings, summaries
```

---

## Paper Naming Convention

Format: `Domain_Year_Publisher_JournalTier_Title.pdf`

Examples:
- `IIoT_2023_IEEE_SCI_BinaryPSO_FeatureSelection.pdf`
- `Cyber_2024_Springer_Q1_IntrusionDetection_Edge.pdf`

See `literature/PAPER_NAMING.md` for full details.
